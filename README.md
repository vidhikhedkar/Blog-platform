Blog Platform Backend

The Blog Platform Backend is a RESTful API built with Node.js, Express.js, and MongoDB. It provides secure user authentication and complete blog management features, allowing users to create posts, interact with content, and engage through comments.

🚀 Features
User Registration and Login
JWT-based Authentication
Create and View Blog Posts
Like and Unlike Posts
Add and View Comments
Protected Routes for Authorized Users
MongoDB Database Integration
RESTful API Architecture

🛠️ Tech Stack
Node.js
Express.js
MongoDB
Mongoose
JWT (JSON Web Token)
bcryptjs
CORS
dotenv

📌 API Modules
Authentication
Register User
Login User
Blog Posts
Create New Post
Get All Posts
Like/Unlike Posts
Comments
Add Comment to a Post
Get Comments for a Post

📂 Project Highlights
Secure authentication using JWT.
User-specific post creation and interactions.
Like and comment functionality for blog posts.
Clean and scalable REST API structure.
MongoDB integration using Mongoose ODM.

Organized project architecture with controllers, models, middleware, and routes.
This backend serves as the API for the Blog Platform application, providing secure authentication, blog management, post interactions, and comment functionality through a scalable and maintainable RESTful architecture.
